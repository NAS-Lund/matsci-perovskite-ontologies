# matsci-perovskite-ontologies
The repository includes two connected ontologies. The first ontology is a general materials-science ontology. It contains concepts and relations reusable across materials-science domains. The other ontology is a perovskite-specific ontology. It contains only concepts and individuals that are specific to perovskite materials.

matsci-ontology is the main ontology, and perovskitemat-ontology is linked to the matsci-ontology. It is important to change the version number of each ontology. And do not forget to change the version number of matsci-ontology in the link in the perovskitemat-ontology file.
